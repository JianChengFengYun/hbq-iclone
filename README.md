> 轻量级项目脚手架  

## 安装   

> npm i -g iclone-cli   
---

## 命令   

### iclone init

> 根据提示创建一个新项目      

### iclone add    

> 添加一条模板配置       

### iclone delete    

> 删除一条模板配置   

### iclone list   
 
> 查看当前模板配置   

### 查看帮助  

>   iclone || iclone --help
